sam布置的第一个作业。
零基础学习前端，之前刷过w3school里的html，css和js教程，对网页设计有一点点的了解，但完全没有实战经验。
这是一个纯静态页面，在写的过程中遇到了许多问题，改了两版，目前这个版本虽然还是很粗糙，但比起第一版还是好了不少的。
总结的问题：
	1、class命名
		一个页面模块有三个部分：header，body，footer
		命名不要带有方向性词汇
	2、class继承
		具有同样特性的元素可以使用相同的类，特殊的部分采用各自的类。例如一些元素共同拥有绿色背景，10px的margin等等，可以在.content定义这些；元素一是黄色字体，元素二是蓝色字体，需要在.content-y 和.content-b里各自声明。
	3、标签的使用要符合语义
	4、选择器层级的考虑
